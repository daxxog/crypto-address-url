# CryptoAddressUrl

  Convert a crypto currency address to a url.

  [![Build Status][travis-image]][travis-url]

Install
-------
stable
```bash
npm install -g crypto-address-url
```
edge
```bash
npm install -g https://github.com/daxxog/crypto-address-url/tarball/master
```
